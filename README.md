# 🎬 Movie Explorer App

A modern, responsive movie exploration application built with vanilla JavaScript, HTML5, and CSS3. Discover popular movies, search for your favorites, and get detailed information about each film.

![Movie Explorer Screenshot](screenshot.png)

## ✨ Features

- **🎯 Popular Movies**: Browse currently popular movies from TMDB
- **🔍 Search Functionality**: Real-time search with debouncing
- **🎭 Filter System**: Filter by genre, year, and rating
- **🌓 Dark/Light Mode**: Toggle between themes with persistent storage
- **📱 Responsive Design**: Works seamlessly on all devices
- **🔄 Infinite Scroll**: Automatically loads more movies as you scroll
- **ℹ️ Movie Details**: View comprehensive information in modal popups
- **⚡ Smooth Animations**: Card animations, transitions, and loading states
- **♿ Accessibility**: ARIA labels, keyboard navigation, and semantic HTML

## 🛠️ Technologies Used

- **HTML5**: Semantic markup, accessibility features
- **CSS3**: Flexbox, Grid, Custom properties, Animations
- **JavaScript**: ES6+, Async/await, Fetch API, Local Storage
- **TMDB API**: Movie database integration
- **Font Awesome**: Icons
- **Google Fonts**: Typography

## 📋 Prerequisites

- A modern web browser
- TMDB API key (free)

## 🚀 Getting Started

### 1. Get a TMDB API Key
1. Visit [TMDB](https://www.themoviedb.org/) and create an account
2. Go to Settings → API
3. Request an API key
4. Copy your API key
